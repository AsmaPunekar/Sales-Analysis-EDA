# 📌 Project Overview

Businesses generate large volumes of sales data every day. Analyzing this data helps organizations understand customer purchasing behavior, identify top-performing regions and products, and make data-driven decisions.

This project performs **Exploratory Data Analysis (EDA)** on fourth-quarter sales data to generate **weekly, monthly, and quarterly business reports** using Python. Interactive visualizations and statistical analysis provide valuable insights into sales performance and business trends.

---

# 🎯 Project Objectives

* Analyze sales performance during the fourth quarter of 2020.
* Generate weekly, monthly, and quarterly sales reports.
* Identify top-performing states, cities, and products.
* Discover sales trends and seasonal patterns.
* Visualize business insights using charts and graphs.

---

# 📂 Dataset

**Dataset:** AusApparalSales4thQrt2020.csv

The dataset contains sales transaction records, including:

* Date
* State
* Product Category
* Sales Revenue
* Units Sold

* Sales Channel
